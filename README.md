# Accessible Flyer Slides

This repository contains an HTML slide deck on creating and sharing accessible flyers.

The slide deck is designed for faculty, classified professionals, managers, and staff who create or share digital flyers through email, Canvas, webpages, SharePoint, social media, or PDF attachments.
