Accessible Flyers

Canva, Email, Canvas, and Web Sharing

Flyers can be visually helpful, but the same information must also be available as accessible text wherever the flyer is shared.

Decorative icon. No information is conveyed by the image alone.

The Core Accessibility Question

If someone cannot see the flyer, can they still access the information?

If the answer is yes, you are on the right path. If the answer is no, the flyer or the way it is shared needs to be fixed before sending, posting, or publishing.

Not Accessible

  • Only posting a flyer image
  • Writing “see attached flyer”
  • Using text trapped inside an image
  • Sending an untagged PDF
  • Using low contrast or crowded layouts

Accessible

  • Provide all details as real text
  • Use meaningful links
  • Attach an accessible PDF when needed
  • Use the flyer image as a visual supplement
  • Mark the flyer decorative when appropriate

Accessible Flyer Workflow

This workflow applies to flyers shared by email, posted online, placed in Canvas, shared on SharePoint, or attached as a PDF.

1. Draft in Word

Type the full event information before opening Canva.

2. Design in Canva

Use accessible design choices and keep the layout simple.

3. Export as PowerPoint

Download from Canva as a Microsoft PowerPoint file, not directly as a PDF.

4. Check Accessibility

Open in PowerPoint and run Review > Check Accessibility.

5. Export Tagged PDF

Create a PDF with document structure tags for accessibility.

6. Share Correctly

Include all event details as accessible text where the flyer is shared.

Step 1: Draft the Information First

Before opening Canva, draft the full event information in Microsoft Word. This gives you a clean text version that can be reused in email, Canvas, SharePoint, webpages, or social media posts.

Include

  • Event title
  • Date and time
  • Location or Zoom information
  • Registration link, including the full URL
  • Contact email
  • Accommodation statement

Accommodation Statement Example

This event is accessible. To request accommodations, contact [email] at least 5 business days in advance.

Do not send or post “See attached flyer” without also including the details in text.

Step 2: Design in Canva Using Accessibility Rules

Canva can be used to create visually appealing flyers, but the design still needs to support readability and access.

Do This

  • Use dark text on a light background
  • Use light text on a dark background when contrast is strong
  • Use simple, readable fonts
  • Keep the layout clean
  • Leave enough spacing between sections

Avoid This

  • Text over busy images
  • Script or decorative fonts
  • Low contrast color combinations
  • Crowded layouts
  • Turning text into an image

The flyer may be visual, but the information should never be trapped inside the image.

Step 3: Export from Canva as PowerPoint

When the flyer design is finished in Canva, do not export directly to PDF. Export the flyer as a PowerPoint file first so you can check and repair accessibility before creating the final PDF.

Canva Export Steps

  1. Select Share.
  2. Select Download.
  3. Choose Microsoft PowerPoint (.pptx).
  4. Download the file.
  5. Open the file in Microsoft PowerPoint.

Do not export directly from Canva to PDF if the PDF needs to be accessible.

Step 4: Check Accessibility in PowerPoint

After opening the Canva-exported PowerPoint file, use PowerPoint’s built-in accessibility checker before creating the PDF.

PowerPoint Steps

  1. Open the file in PowerPoint.
  2. Go to Review.
  3. Select Check Accessibility.
  4. Fix the issues listed.

Check For

  • Slide title
  • Logical reading order
  • Readable text
  • Sufficient contrast
  • Alt text for meaningful images

Confirm the content reads in a logical top-to-bottom order.

Step 5: Export as an Accessible PDF

After fixing accessibility issues in PowerPoint, export the file as a tagged PDF. Tags help screen readers understand the structure of the document.

PowerPoint PDF Export Steps

  1. Go to File.
  2. Select Export.
  3. Select Create PDF/XPS.
  4. Select Options.
  5. Check Document structure tags for accessibility.
  6. Export the PDF.

Do not use “Print to PDF.” Printing to PDF can remove accessibility tags.

Step 6: Share Correctly

When sharing a flyer, treat the flyer as a visual supplement. The full event information should also appear as real text in the place where the flyer is posted or sent.

Include as Text

  • Event title
  • Date and time
  • Location or Zoom information
  • Meaningful registration link
  • Contact information
  • Accommodation statement

Use This In

  • Email messages
  • Canvas pages
  • Webpages
  • SharePoint pages
  • Social media posts
  • Online event listings

Never rely on the flyer image alone to communicate event information.

Flyers in Email

When sending a flyer by email, do not rely on the flyer image alone. Add all event information as accessible text in the body of the email first, then place the flyer image below as a visual reference.

Recommended Email Workflow

  1. Download the flyer as an image file, such as PNG or JPG.
  2. Open a new email message.
  3. Type or paste all event information as accessible text at the top of the email.
  4. Use short paragraphs, lists, and meaningful links.
  5. Insert the flyer image below the accessible text.
  6. Mark the flyer image as decorative, if your email platform allows it.
  7. Attach the accessible PDF version, if needed.
  8. Send a test email to yourself and check it on desktop and mobile.

Put the information in text first. Place the flyer image below. Mark the flyer as decorative when possible.

Canvas Announcements: Flyer Image + Accessible Text

If you post a flyer in a Canvas Announcement, the flyer should be treated as a visual supplement. The full message must be provided as accessible text above the image.

Recommended Workflow

  1. Download the flyer as an image file, such as PNG or JPG.
  2. Open the Canvas Announcement editor.
  3. Add the full event information at the top using real text.
  4. Use clear headings, short paragraphs, lists, and meaningful links.
  5. Insert the flyer image below the accessible text.
  6. Select the flyer image and open Image Options.
  7. Check Decorative Image.
  8. Save or post the announcement.

Do not make people open or read the image to get the event details.

Decorative

If the flyer is fully repeated in nearby text, mark it decorative.

Informational

If the image adds meaning, write short alt text that explains its purpose.

Complex

If the image includes detailed information, provide the details as nearby text.

Quick Accessibility Checklist

  • The flyer information is available as real text wherever the flyer is shared.
  • The flyer uses readable fonts and strong color contrast.
  • The flyer does not place important text over a busy image.
  • The PowerPoint file has been checked with the Accessibility Checker.
  • The PDF is exported with document structure tags for accessibility.
  • The email, Canvas page, or webpage uses meaningful links.
  • The flyer image is marked decorative when the same information appears nearby as text.
  • The message works on desktop and mobile.

Accessible sharing means the message is not trapped inside the image.

Why This Matters

Perceivable

The information is available as text, not only as an image.

Operable

Links and page content can be reached by keyboard and assistive technology.

Understandable

Event details are organized clearly with headings, lists, and plain language.

Robust

Real text is more reliable across browsers, devices, screen readers, magnification tools, and mobile settings.

Final Reminder

If someone cannot see the flyer, can they still access the information?

The accessible workflow is not just about making the flyer file better. It is about making sure the event information is available wherever the flyer is shared.

The flyer can be visual, but the information must also be available as accessible text in the email, webpage, Canvas page, social media post, or other location where it is shared.
